# ProjektowanieAplikacjiUI

## Jak uruchomić

Ponieważ ta aplikacja korzysta z **ES Modules**, do działania wymaga lokalnego serwera WWW.

### Zalecena użyć Pythona:
Do uruchomienia tego trzeba mieć zainstalowanego Pythona (wersja obojętna):

1. Pobierz folder z projektem - "AplikacjaRandkowa_AURA.
2. Otwórz **Terminal** lub **PowerShell** w folderze projektu (np. `d:\newworkspace`).
3. Przejdź do folderu z projektem np. D:\AplikacjaRandkowa_AURA
4. Wpisz poniższe polecenie, aby uruchomić prosty serwer:
   ```bash
   python -m http.server
   ```
5. Wejdz na przeglądarce na stronę:
```bash
    http://localhost:8000
```
